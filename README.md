## Installation

- All the `code` required to get started


### Clone

- Clone this repo to your local machine using `https://github.com/giorgos321/React_Udemy_Clone`
```shell
$ git clone `https://github.com/giorgos321/React_Udemy_Clone`
```

### Setup
> update and install the dependencies 

```shell
$ npm install 
```

> now use npm start to start the project

```shell
$ npm start
```
